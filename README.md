# Tribe OS 🏹
**The Agentic Business Operating System for the Global South.**

Tribe OS is an open-source, mobile-first management framework designed to help small businesses in developing nations transition from paper-ledgers to AI-driven operations. 

### 🌟 Vision
Most "SaaS" tools are built for Western markets with high monthly costs and complex UIs. Tribe OS is built for the **"WhatsApp Economy,"** allowing business owners to manage their entire operation through voice, images, and text.

### 🛠️ Core Features
- **Invisible CRM:** WhatsApp-based member registration and payment tracking.
- **Agentic Sync:** Multi-agent workflows that bridge the gap between offline cash transactions and online dashboards.
- **Data Sovereignty:** Built-in "Redundancy Engines" that mirror data to Google Sheets and local backups for 100% reliability.

### 🧪 Technical Stack
- **Framework:** Next.js (Dashboard) & n8n (Automation Brain)
- **Intelligence:** Claude 4.7 Opus (Lead Architect) & Gemini 1.5 (Vision/Voice)
- **Database:** Supabase & PostgreSQL
- **Bridge:** Evolution API (WhatsApp)
